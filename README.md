# VMolberte
Приложение для визуализации данных.

# Минимальные системные требования
+ *ОС:* Windows 10, Windows 11;
+ *Объём ОЗУ:* 1024MB;
+ *Место на диске:* 600MB;
+ Подключение к сети интернет при работе с приложением и просмотре отчётов.
